# Lesson: String Fundamentals

## Warm-up Written Question (15 minutes)

1. String Creation

2. String Properties

   - Immutability
   - Unicode support

3. Common String Methods
   - charAt()
   - substring()
   - slice()
   - indexOf()
   - replace()
   - split()
   - toLowerCase()/toUpperCase()
